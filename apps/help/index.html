How to add an app

First off, navigate to apps/

The directory should have 3 folders. chrome, help and sample. Add a new folder and name it how you want to name your app.

Add an index.html file. This is the starting point of your application. Additionally you will need to add a icon.png file. This is the icon that will be displayed on the desktop.

NOTE: The icon.png file should have a 1:1 aspect ratio. This means that the width and height

Now, go back to the main directory and open the apps.json file. Add your app to the list of apps.

Ta-Da. You now have an app!
